🎹 Interstellar Piano

Hardware Synthesizer with Web Dashboard

Project Overview

The Interstellar Piano is a multi-microcontroller embedded system designed to bridge digital electronics with web development. It allows users to play notes, execute complex chords, and trigger pre-programmed melodies (including Hans Zimmer's iconic Cornfield Chase) using either a physical 4x4 matrix keypad or a remote web dashboard.

System Architecture

Arduino Mega 2560

The musical core. It handles physical input from the keypad, generates frequencies for the passive buzzer via hardware timers, and controls an array of 8 LEDs for real-time visual feedback.

ESP32-S3

The communication hub. It acts as a Wi-Fi Access Point and hosts an asynchronous web server. It provides the browser UI and communicates via UART with the Mega at 115200 baud.

Hardware Requirements

  • 1x Arduino Mega 2560
  • 1x ESP32-S3 (or standard ESP32)
  • 1x 4x4 Matrix Membrane Keypad
  • 1x Passive Buzzer
  • 8x LEDs (Various colors recommended)
  • 8x 220Ω Resistors (for LEDs)
  • Jumper wires and Breadboard

Engineering Note: The Arduino Mega operates at 5V logic, while the ESP32 is a 3.3V device. While connecting them directly often works for prototyping, it is highly recommended to use a logic level converter or a simple voltage divider (using resistors) on the Mega TX -> ESP32 RX line to protect the ESP32 pins.

Peripherals (Connected to Arduino Mega)

  • Buzzer: Pin 8 (Signal) & GND
  • Keypad Rows (1-4): Pins 62, 63, 64, 65 (A8 to A11)
  • Keypad Cols (1-4): Pins 66, 67, 68, 69 (A12 to A15)
  • Note LEDs (via 220Ω resistors): Pins 22, 24, 26, 28, 30, 32, 34, 36

Communication Protocol

The two microcontrollers communicate bidirectionally by exchanging formatted string packets over serial:

  • Mega -> ESP32: N,C4,261.63 (Updates the web UI with the current note)
  • Mega -> ESP32: C,Amin (Updates the web UI with the current chord)
  • ESP32 -> Mega: 1, E, F, D (Sends execution triggers generated by web user clicks)
